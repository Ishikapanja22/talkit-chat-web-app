# TalkIt — Real-Time Chat App

A real-time chat web app built with the MEAN stack and Socket.IO.

## Live Demo
(https://talkit-chat-web-app.vercel.app/) 

## Preview
chat preview (https://github.com/Ishikapanja22/talkit-chat-web-app/blob/main/chat%20preview.png)

## Features
- User Registration & Login
- Real-time messaging with Socket.IO
- Multiple chat rooms
- Responsive UI

##  Tech Stack
- Frontend : Angular
- Backend : Node.js + Express
- Database : MongoDB
- Socket.IO Client
- Socket.IO
- Mongoose 

##  Getting Started
### Backend
```bash
cd backend
npm install
npm start
```
### Frontend
```bash
cd frontend
npm install
ng serve
```
